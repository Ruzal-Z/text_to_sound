
Python script with which text can be played with sound.

1. For this script to work, you need to install gTTS (pip install gtts) and playsound (pip install playsound).
2. We write the desired text in the field 6. text = "Here we write the text for playback !!!".
3. Run the script.
4. An mp3 file will appear in the folder with the script.

Скрипт на Python с помощью которого текст можно воспроизвести звуком. 

1. Для работы данного скрипта необходимо установить gTTS (pip install gtts) и playsound (pip install playsound).
2. Пишем нужный текст в поле 6. text = "Здесь пишется текст для воспроизведения!!!".
3. Запускаем скрипт.
4. В папке со скриптом появится файл mp3.
